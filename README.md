# CS370
Briefly explain the work that you did on this project: What code were you given? What code did you create yourself?
For this project I was given the base framework for the gameexperience class as well as the maze class. A lot of the set up for the agent was also included for the
various libraries as well as setting up the different initial environment variables. What I wrote was the code for the agent actively running through the maze
and the process by which the agent can learn to navigate better. Also the code for printing out the results of the various runs.
What do computer scientists do and why does it matter?
Computer scientests create programs to fulfill needs of a client. It matters because there need to be people who are able to take the needs of a client
and then creating a program to be able to fulfill those needs and make it future proof as much as possible.
How do I approach a problem as a computer scientist?
Usually by trying to just work bit by bit and try to make sure the resulting code will do what it needs to and then work on connecting it to the other parts of
the program as I progress.
What are my ethical responsibilities to the end user and the organization?
Creating programs that fulfill the needs of a user and will continue to do so as long as possible. Also ensuring that a program made is secure and can't be
maliciously used or corrupted for malicious use.
